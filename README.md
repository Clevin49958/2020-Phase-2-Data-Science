# msa-data-2020-phase-2

# How to set up project

For windows
```shell
python -m venv venv
.\venv\Scripts\activate
pip install -r requirements.txt
jupyter notebook
```

For Mac (from memory I haven't checked let me know if there are issues)
```shell
python3 -m venv venv
source ./venv/bin/activate
pip install -r requirements.txt
jupyter notebook
```
